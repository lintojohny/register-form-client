# Register Form

## Overview

This repository contains the front-end implementation for a user registration form.

## Setup Commands

1. Install dependencies:

    ```bash
    npm install
    ```

2. Run test cases:

    ```bash
    npm run test
    ```

3. Start the app:

    ```bash
    npm run dev
    ```

## Tools Used

- **React:** v18.2
- **Jest:** Test framework
